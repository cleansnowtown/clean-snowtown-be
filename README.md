# CleanSnowTown-v2.0 (2023.11~)

## 개요 
기존에 작업했던 [CleanSnowTown](https://github.com/yumi530/CleanSnowtown)을 개선해보기 위한 프로젝트입니다.
프론트는 react 기반, 백엔드는 rest api 방식으로 전환하고 각자 자유롭게 다양한 기능을 테스트할 수 있게 작업을 진행할 예정입니다.

## 기술 스택

## 프로젝트 설치 및 실행 방법
### 프론트
```
cd frontend
npm start
```

### 백엔드
```
cd backend
```
